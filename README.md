# 到梦空间活动捡漏助手

## 库
1. `Requests`: 完成爬虫部分
2. `PyQt5`: 编写Ui界面（部分AI完成）

## 原理
- 到梦空间的活动可以通过分享链接拿到活动页面网址，而网址上会显示报名人数和总人数
- 通过requests爬取报名信息即可实现蹲守捡漏，空出名额后程序通过发声提醒
